Waveform Display

How to render the energy_curve string — each option uses identical data

A — Mirror Serato-style
Deck A  ▶ bar 24 / 128

Symmetric bars above + below center

Classic professional DJ waveform look. Feels technical, balanced. Playhead splits past (dimmed) from upcoming (bright).

B — Gradient Fill Smooth / Modern
Deck A  ▶ bar 24 / 128

Smooth filled curve with gradient fade

Less clinical, more musical. The energy shape reads as a landscape. Good for seeing build-ups and drops at a glance.

C — Dot Matrix Minimal / Data-art
Deck A  ▶ bar 24 / 128

Grid of dots — height = energy level

Looks like an LED matrix or oscilloscope. Each column is one bar, each dot represents an energy unit. Unique, hardware-inspired.

D — A+B Overlap Transition-first
Deck A fading → Deck B incoming  transition zone

Both deck waveforms overlaid in one strip

Shows the crossfade zone where A and B overlap. Orange = A fading out, Blue = B fading in. Makes the transition tangible.

All four options use the same energy_curve data from the track analysis
